# 一个图书管理系统

================================================================================

## 1.介绍
这是一个无经验的学生开发的一个简单的前端项目，它可以被应用在图书馆的借、还书场景中。它集结了对图书、用户的增删改查、对借阅记录的查询以及网站公告功能。不过该项目只用于学习之用。

### 1.1用户功能介绍
用户可以创建用户账户、登录、更改账户密码、更改部分账户信息（联络方式）、根据模糊查询和类别和语言查询图书、查看网站公告、借书、还书、查询自己的借阅记录。其中借书是有额度的，额度是跟据用户等级决定的，一开始用户等级是1，可借5本书，并且借书后书的库存会减少，还书后书的库存会相应增加。

### 1.2管理员功能介绍
管理员可以发布和更改网站公告、对用户信息删改查、对图书信息增删改查、查阅所有用户的借阅记录。其中网站公告可以根据日期查询，用户信息可以模糊查询、图书信息可以模糊查询、借阅记录可以根据用户信息和图书信息模糊查询，并且支持已还和未还记录分开查询。


## 2.部署说明

### 2.1requirements(测试环境)
1.Mysql 8.0
2.Apache 2.4.54 x64
3.任一主流最新版本浏览器（Edge、Chorme、Safari、Firefox、IE）

### 2.2部署
用Mysql执行books_ma.sql，将index.html所在文件夹放到服务器主页，然后用浏览器访问服务器主页即可。（执行数据库初始化这一步会在后续版本优化）


## 3.使用说明

### 3.1用户界面
主页为index.html，如果未登录则只能查询图书信息、看网站公告，进行登录或注册行为后则可以点击导航栏借书还书和查询自己的借阅记录了，并且可以点击右上角自己的用户名更改信息，或者点击注销来退出登录。若借书不满足相应条件则会弹出相应的警告信息。

### 3.2管理员

### 3.2.1管理员的注册
目前新增加管理员只能用Mysql执行，更改相应Mysql用户的等级为0即可以把该用户变为管理员。

### 3.2.2管理员界面
在登录界面登录后若账户是管理员则直接跳转到管理员界面。管理员界面分为用户管理、书籍管理、公告管理、借阅查询界面，可以通过导航栏跳转。用户管理界面还可以重置用户密码。并且也可以点击右上角用户名改密码或者点击注销退出登录。
